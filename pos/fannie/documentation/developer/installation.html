Installing Fannie is supposed to be easy.
  1. Drop it on a webserver
  2. Navigate to the install.php script with a browser
    • The install script tries to write the configuration file, config.php. It'll tell you what the web server's uid is and whether or not the config file is writable. If it isn't, you need to fix that.
    • Once you plug in your database connection values, the install script creates all the tables and views it needs (in theory).
  3. There is no step 3
